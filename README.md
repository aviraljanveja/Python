- This Python course is an abridged version of [MIT 6.0001 : Introduction to Computer Science and Programming in Python](https://ocw.mit.edu/courses/6-0001-introduction-to-computer-science-and-programming-in-python-fall-2016/). 

- The 12 video lectures from the original course have been simplified and re-organized into 8 written chapters that can be followed at your own pace.

- The **Chapters** folder contains course content in PDF format. The **Code** folder contains example programs with comments, covering everything from fundamentals to advanced stuff. 

- It is highly recommended that learners make use of [pythontutor.com](https://pythontutor.com/python-compiler.html) which would allow them to visually go through what really happens **behind the scenes** in a Python interpreter.
